# Age Calculator (JavaScript)

A simple age calculator built with HTML, CSS and JavaScript. This project includes both separate files (HTML, CSS, JS) and a combined single-file example.

## Files

- [AgeCalculator.html](https://github.com/asrith1999/CodeAlpha_Project_AgeCalculator/blob/main/AgeCalculator.html) — Main HTML page (uses external CSS and JS files)
- [AgeCalculatorcss.css](https://github.com/asrith1999/CodeAlpha_Project_AgeCalculator/blob/main/AgeCalculatorcss.css) — Styling for the calculator
- [AGE.js](https://github.com/asrith1999/CodeAlpha_Project_AgeCalculator/blob/main/AGE.js) — JavaScript logic to calculate age
- [javascript age calculator.html](https://github.com/asrith1999/CodeAlpha_Project_AgeCalculator/blob/main/javascript%20age%20calculator.html) — Single-file HTML with embedded styles and script

## How to use

1. Clone or download the repository.
2. Open one of the HTML files in your browser (for example, `AgeCalculator.html` or `javascript age calculator.html`).
3. Enter your date of birth and click the calculate button to see your age.
